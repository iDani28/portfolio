# Portfolio personal - Daniel Moreno Ruiz

Portfolio personal desarrollado con Next.js, React y Tailwind CSS. Incluye secciones de presentación, formación, experiencia, proyectos, skills y contacto con integración de EmailJS.

## Tecnologías

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- EmailJS

## Web en producción

https://portfolio-khaki-beta-25.vercel.app

## Variables de entorno

Copia `.env.example` a `.env.local` y rellena los valores:

```
NEXT_PUBLIC_EMAILJS_SERVICE_ID=
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=
```

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.
